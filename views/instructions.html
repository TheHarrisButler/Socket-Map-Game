How To Play

Welcome to The Map Game! A game where you and your opponents send eachother a google maps street view of an unknown location. The goal is to guess what US state that location is in. You get one point for each correct guess. First one to three points wins! The following is a complete step-by-step guide to understanding how to play. This guide will walk you through: creating a game, joining a game, submitting an address, and finally submittting a guess. If you want to learn a little more about how this game was created or are just wondering what the catylist for creating this game was, please head over to the blog tab. I hope you enjoy!

Step 1: Creating a game

If you haven't yet been invited to join a game (i.e. you aren't very popular) you must first create a game. From the home page, navigate to the hamburger menu at the top of your screen (hint: its the same hamburger menu you used to get to these amazing instructions). From there click the "Create Game" button. A unique game id will appear bellow the join game button. Send this id to one of your friends so they can join your newly created game!

Step 2: Join a game

After creating your new game, a unique game id will appear below the "Join Game" input, simply paste the game id into the form and click the "Join Game" button to join the lobby. If, however you were sent a game id from a friend simply paste the id into the "Join Game" form and click the "Join Game" button. After joining the lobby you should see you player name and current score in the players card. The name and color is based on the time you joined the game. The first too join gets green and the second to join gets red. Awesome! You are now ready to start the game!

Step 3: Submitting an address

If you are the game creator then you start the game by submitting an address. Simply input a desired address in the "Address Form" and click submit. From there the address is sent to your opponent in the form of a google maps street view (the street view interface only changes upon recieving an address from an opponent not after submitting an address). While it would be cool to send a google street view of the Death Star or Mars as of right now only valid United States addresses are accepted.

  • Valid United States addresses only
  • Street address cannot contain appartment or office numbers (i.e. 3800 N Lamar Blvd #220)

Step 4: Submit a guess

After your opponent sends an address, your google maps street view interface will update to the appropriate location. From there you are free to move around and find clues as to which state the address is located in. Once you feel you are ready to make a guess click the drop down menu and select a state. If you are correct you will see your score increase by one and a congratulatory message. Remember the first player to 3 points wins!
